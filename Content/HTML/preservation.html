On the Preservation tab, you can select the locations where you can put, for example, email on hold. These locations have been previously defined by your Administrator via the Subscription option in the menu.

Edit Preservation Settings

  1. In Draft state, select Edit Preservation Settings.


  2. Select the locations from which data needs to be preserved.
    Define the Start and End Date. 

  3. Click Save.



    When defined, you will see an overview of all locations from which data will be preserved. 

    Each location is displayed on a card on which the number of custodians that will or will not be preserved is shown.
    By default, all custodians are added to each location to secure their data. Via the Custodian Details, you can exclude one or more custodians per location.

    Select any of the blue links in each card to show the Custodian Details and filter the grid at the same time (for example, click the number behind Remaining to only view the remaining preservations for that location).
    Exclude multiple custodian/location-combinations via the button in the top left corner, or exclude a single custodian/location-combination via the Exclude Custodian Location button at the end of the row.




    At the bottom of each card you can view the results (Done, Failed or Remaining).

    With the icons in the top right corner of each card you can see if the preservation is done manually or automatically (with an in-place hold). With automatic preservation, the system will secure the data. With manual preservation, a person must perform an action (for example, create a forensic image of a laptop). Locations that are manually preserved (secured), can be marked as such via the menu or the button Set Custodian Location status to Preserved at the end of the row in the grid.
