Supported Browsers

Supported Browsers for Custodian Portal

All modern and up-to-date browsers are supported. The custodian portal can be accessed on a variety of devices.

Supported Browsers for Legal Hold App

All modern and up-to-date browsers are supported, including:

  • Google Chrome

  • Apple Safari (MacOS)

  • Microsoft Edge

  • Mozilla Firefox

Minimum of 2GB RAM or 4GB RAM is required for users using the Analytics functionality.

Minimum resolution is 1024 x 768, the recommended resolution is 1920 x 1080.

It is recommended to keep your browser updated.

Note on Internet Explorer

Internet Explorer 11 is not supported because the Edge New browser is available for enterprises. Individuals can download Edge New also. Edge New has IE11 compatibility mode, which allows the use of legacy applications.
