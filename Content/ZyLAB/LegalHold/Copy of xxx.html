Locations

The Locations tab allows you to add locations that need to be preserved in the hold. Please note that for in-place hold preservation, you need to set up an integration with the Microsoft 365 Compliance Center first.

Create Location

  1. Select .

  2. Define the Name, Preservation Type, Source and Tenant(s).

    • For example, Email 365 (Name), Microsoft 365 Compliance Center (Preservation Type) and Email (Source) for in-place preservation.

    • Or, Laptop (Name) and Manual (Preservation Type). The Preservation Type 'Manual' allows Forensic Investigators or IT Administrator to manually preserve the data.

    Note: Please make sure that custodians from your Azure Active Directory are synchronized and up to date, so that the Legal Hold is aware of the correct OneDrive locations.

  3. Click Save.
    Locations are used when creating a hold to define the data that needs to be preserved. 
    For more information, please refer to Preservation.
