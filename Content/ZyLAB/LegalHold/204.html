Questionnaire Templates

The Questionnaire Templates page contains all Questionnaire Templates of a hold. Questionnaires are used to specify the subjects and sources for collection. 

View, filter, sort, create, edit, delete or export Questionnaire Templates to be sent with the Hold Notice.

  • View created Questionnaire Templates and filter/sort them via the columns.

    Note: Filtering/sorting works the same on all pages (example below is of the Hold Management page).

  • Via the Settings icon  add/remove columns or export the Questionnaire Templates View as CSV, Excel or PDF

  • Edit a Questionnaire Template via the Edit Questionnaire Template icon . 

  • Duplicate a Questionnaire Template via the Clone Questionnaire Template icon . 

  • Delete a single Questionnaire Template via the Delete Questionnaire Template icon . 

  • Delete multiple Questionnaire Templates by selecting them and clicking the Delete selected Questionnaire Templates icon .

  • Update the Questionnaire Templates View with the Refresh icon .
