Settings

Edit the Default Notification Settings

The Settings tab enables the Global Hold manager to set the notification schedule according to their company procedures.

  1. From the menu on the left, select Admin Center > Subscription > Settings.

  2. From the Default Hold Message Settings window, select Edit.

  3. Change the Schedule Settings to your needs.

  4. Click Save.

Select Email Integration

As an Administrator you can now use the option to select which email integration is allowed to be used by the users.

  1. From the Allowed Email Integrations window, select Edit.

  2. Check the box to select the Email Integrations you wish to use.

  3. When more than one integration is selected, click the down arrow to set your Default Email Integration.

  4. Click Save.
