Available Reports

The following reports can be generated:

  • Active Holds Summary Report
    Generates a summary report on the drafts and issued holds.
  • Matter Summary Report
    Generates a summary report on selected matter(s)/hold status(es).
  • Matter Detailed Report
    Generates a detailed report on selected matter(s)/hold status(es).
  • Hold Summary Report
    Generates a summary report on selected hold(s).
  • Hold Detailed Report
    Generates a detailed report on one selected hold.
  • Custodian Report
    Generates a report on selected custodian(s)/hold status(es).
  • Hold Owner Report
    Generates a report on selected hold owner(s).
  • Managing Attorney Report
    Generates a report on selected managing attorney(s).
  • Custodian Preservation Report
    Generates a report on custodians in preservations.
  • Hold Preservation Summary Report
    Generates a report on selected preservations.
  • Hold Preservation Detailed Report
    Generates a detailed report on one selected preservation.
  • Custodian Profile Change Report
    Generates a report on changes in a custodians profile.
  • Custodian Leaving Report
    Generates a report on leaving information of a custodian.
