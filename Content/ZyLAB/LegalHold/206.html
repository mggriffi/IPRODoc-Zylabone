Message Templates

The Message Templates page contains all message (email) templates of a hold. The contents of a hold notice can vary based upon a number of factors including the scope of litigation, the way data is stored, the types of accessible data, and how types of data should be preserved. Several emails that address those factors should be prepared (the first Hold Notice, a Reminder, Escalation, Awareness Reminder and Lift Notice).

View, filter, sort, create, edit, delete or export Message Templates to be sent with the Hold Notice.

  • View created message templates and filter/sort them via the columns.

  • Via the Settings icon  add/remove columns or export the Message Templates View as CSV, Excel or PDF.

  • Edit a Message Template via the Edit Message Template icon .

  • Preview a Message Template via the Preview Message Template icon  .

    • Select a Hold, a Custodian and a device.

  • Duplicate a Message Template via the Clone Message Template icon  .

  • Delete a single Message Template via the Delete Message Template icon .

  • Delete multiple Message Templates by selecting them and clicking the Delete selected Templates icon .

  • Update the Message Templates View with the Refresh icon .
