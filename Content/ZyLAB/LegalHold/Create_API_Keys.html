Create API Keys

IPRO's Public API gives users control over functionality directly from within their own application/administrative portal. Public APIs provide access to the configuration and orchestration of certain tasks, thereby enabling external developers to control some of the product's functionality required for workflow automation.

The API tab allows you to create your own API keys.

Note: You must have system administrator rights to create API keys. API keys are needed to access the API.

  1. Open Admin Center>Subscriptions.

  2. Click .

  3. Click .

    1. Enter a Name.

    2. Set an Expiration date.

    3. Scope will default to only available scope -Read Reports.

    4. Click Save.

  4. The API Key will be created. Copy the API key and click Close.

    Warning: Make sure you copy the API key now as it is not stored and you will not be able to see it again.

  5. The API window will appear . You will be able to view and filter the Name, Key Value, Scope, Active status, Created By and Expiration date.

  6. API keys can be deleted by clicking at the end of the row. You can also select one or more keys and click the Delete icon at the top of the window.

To view the public Legal Hold API, click the link located at the top of the window.
