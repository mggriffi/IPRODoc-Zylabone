Locations

The Locations tab allows you to add locations that need to be preserved in the hold. Please note that for in-place hold preservation, you need to set up an integration with Microsoft Purview first.

Create Location

  1. Select Create Location.

  2. Define the Name, Preservation Type, Source and Tenant(s).

    • For example, Email 365 (Name), Microsoft Purview (Preservation Type) and Mailbox (Source) for in-place preservation.

    • Or, Laptop (Name) and Manual (Preservation Type). The Preservation Type 'Manual' allows Forensic Investigators or IT Administrator to manually preserve the data.

Note: Please make sure that custodians from your Microsoft Entra ID are synchronized and up to date, so that the Legal Hold is aware of the correct OneDrive locations.

Click Save.

Locations are used when creating a hold to define the data that needs to be preserved. 
For more information, please refer to Preservation.
