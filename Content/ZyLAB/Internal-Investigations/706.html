Audit Log

ZyLAB ONE keeps an audit log of all actions that have been taken in a matter, which helps track down mistakes and issues. View all logged events on the Audit Log overview page. A CSV-file of the Audit Log list can be downloaded.

An audit log is not commonly included in a report of an internal investigation, but can be useful for transparency and accountability. If you would like to add an audit log to your report, see instructions below.
