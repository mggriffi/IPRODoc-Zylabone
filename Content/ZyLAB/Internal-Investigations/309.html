User Access

Are you the only user? Then you don’t have to set any user roles. Setting up user roles is not mandatory, but we do recommend it if you plan to use ZyLAB ONE more often. To prevent mistakes, it is useful to give certain users limited access to documents.

 

Example: User Access

Investigations bureau X has one main investigator who searches for potentially relevant documents by using Batch search and Technology Assisted Review. The other investigators on the case merely review these documents on their relevancy. In the situation described above, it’s useful to give the other investigators permission to only review the documents given to them by the main investigator. By adding extra security in Document Security, you can limit access to documents. This makes it easier to adhere to the principle of proportionality.  

There are 4 default user roles available in ZyLAB ONE:

  • Administrator
  • User
  • External User
  • Uploader
