Consumption Overview

View Insights usage statistics.

Instructions

  1. On the Home page, select Insights.


  2. Select Consumption Overview.


  3. View usage per day/week/month/year.
    Select a date range.

  4. View total characters consumed.

    Displays your consumption; The total number of characters consumed during the defined date range.


  5. View usage per extractor.

    Plus an export option to Excel or CSV.
