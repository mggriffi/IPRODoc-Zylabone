Create a New Search

  1. Open ZyLab One. Select your Matter.

  2. From the Home page, select the tile.

    The Search window opens.

  3. Click Start New to create a new search.

  4. Enter a Name for the search. This name will be included in the upload name, so being specific can help identification later in your workflow. Entering a description is optional.

  5. Click Continue.

  6. Once the search is created, you will see Volume (GB) and File Count next to the search name. This is your custodian scope volume and file count, which will update as you refine your search.
