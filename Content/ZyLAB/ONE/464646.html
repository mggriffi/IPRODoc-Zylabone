ZyLAB Collector

The ZyLAB Collector enables you to copy (export) data from one matter to another. Documents and metadata (fields / information about the documents) will be collected and exported.

Instructions

  1. On the Home page, select Upload:

  2. On the Upload page, select Upload > ZyLAB Collector.


  3. In the first step of the Legal Review Collector wizard (Settings), define: 
    • Sources
      Select the source matter(s) you want to collect data from.
    • Family Expansion
      Select the mode in which documents must be collected. For now, only 'None (only matching documents)' is available.
    • Impersonate Service User
      It is advised to keep this setting enabled.
    • Set the Custodian and Source for all locations
      Define (select or add) the custodian and source.
  4. Click Next.
  5. Define the Mapped and/or Not Mapped fields (these fields contain the metadata).
    Most fields will be mapped automatically and saved as Mapped fields. All remaining fields can be mapped manually.
  6. Select Start Data Collection.
