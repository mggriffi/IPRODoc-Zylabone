How to Set Up SSO with Microsoft Entra ID

The following instructions provide steps on how to set up SSO in ZyLAB One with your own Microsoft Entra ID identity provider.

  1. Sign in to portal.azure.com and go to Microsoft Entra ID.

  2. Cick on App Registrations.

  3. Click on New registration.

  4. Choose a name for your application (you’re free to choose whatever you want).

  5. Choose if this should be single tenant or multitenant (this is your choice).

  6. Enter the redirect URL that ZyLAB provided to you. Please know that this is CASE sensitive.

  7. Click Register.

  8. Open up the app registration and go to Authentication.

  9. Enure that ID Tokens are enabled.

  10. Go back to Microsoft Entra ID.

  11. Go to Enterprise Applications.

  12. Select the newly created ZyLAB One application and click on permissions.

  13. It's advised to grant admin consent for your entire company. This avoids confusion for users who otherwise will get prompted to give their consent to sign in.

  14. Click on Accept.

  15. You should see a notification that this worked.

  16. Provide ZyLAB with the following details:

    1. Application (Client) ID

    2. Object ID

    3. Directory (tentant) ID
