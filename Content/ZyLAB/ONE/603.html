Step 3: Create Review Batches

Prerequisites

  • A project has been created. The Initial Training Set of one or more issues has been reviewed by a reviewer/Subject Matter Expert.

Instructions

  1. In Assisted Review, create a New Training Batch for each reviewed Initial Training Set. The size depends on the number of available reviewers.
  2. Open the Home page, select Browse.
  3. In Facet View, select the new training batch that has been created. 
  4. Open the Training Batch in the Document List.
  5. Add the ranking field for the Training Batch.
  6. Sort on the ranking field with the highest ranking on top (Descending).
  7. Now, based on this sorting order, create Review Batches.
  8. Divide the sub-batches over all reviewers. Add the Review Batches column to sort on the different sub-batches.
  9. Inform all reviewers which Review Batch to review.
  10. Repeat step 1 to 9 until all documents are reviewed.
