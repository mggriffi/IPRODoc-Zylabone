Step 2: Assign and Evaluate Initial Training Se

Prerequisites

  • A project with one or more issues has been created.

Instructions

  1. From the navigation tree, select Home.

  2. Select Browse. 

  3. The Facets overview will appear.
  4. Scroll to the facet: Assisted Review Training Batches.
    The first Training Batch of an issue is similar to the Initial Training Set.
  5. Select the correct Training Batch (Initial Training Set) in the facet.
  6. Select Open in Document List. 
  7. Copy the URL in an email to your reviewer or Subject Matter Expert. 

    Note: The documents should be tagged with the tags associated with the issue. So, for the issue/initial training set "matisse", the tag "matisse - NR" and "matisse - R" should be used (where NR = Not Responsive and R = Responsive).

  8. After review, the results are evaluated in the Assisted Review Progress tab.
  9. Continue with Create Review Batches.
