Step 2: Review Topics and Create Issues

Prerequisites

  • A project with topic modeling and no issues has been created.

Instructions

  1. Open the Topic Modeling tab and select a topic.

  2. To view the documents in this topic, select Document List.

  3. Review the documents in this topic.
  4. Return to Assisted Review.

  5. Create an issue for topics that are identified as responsive.


  6. Define the issue name, add a description for the issue and enter the batch size (size of the initial training set):


  7. Click Save.
  8. Repeat step 1 to 7 for other topics that are displayed in the Topic Modeling tab.
