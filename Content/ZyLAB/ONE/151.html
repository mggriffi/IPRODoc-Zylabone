Columns

You can (de)select the columns (tags and/or metadata fields) that are visible in the Document List.

Selecting Columns

  • In the Document List, select the Settings icon 

  • Select Columns

  • Select the checkboxes of the columns you want to see. You can use the filter to search for (a) specific column(s).

  • (Optional) Once you have selected the desired columns, you can sort them. 

Available Columns

If you are performing a Keyword Search, you can select the Hit Count and Search Relevancy Rating columns to preview the number of hits and the relative density of hits in the document.

  • Hit Count: Number of hits (total) found in the document.

    Click on the Hit Count number to quickly navigate to the hits found.

  • Search Relevancy Rating: Number of hits (total) found in relation to the document (file) size.

    For example: a one-page document with one hit has a higher Search Relevancy than a hundred-page document with one hit.

    Note: Hits are found in the text processed. The Hit Count displays the total number of hits found. If a document has multiple Views - for instance, PDF and OCR Views - then it is possible that not all hits are not shown in the selected View, because the hits might not be visible in this View.
