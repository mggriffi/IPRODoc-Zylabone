Default Matter Roles

All default matter role permissions are listed in the first column of the table below. Remember, permissions are added to roles (not users).

The default matter roles are: Users, Users External, Users Managers and Users Upload.

Roles are assigned to users/groups (in SaaS with b2c authentication no groups are used). 

The permissions listed below, only apply to the role types Matter and Current Matter. This means that not all permissions are listed here. For an overview of all permissions, see Role Permissions Explained.
