Entity Management

View a list of all extraction types. Per extraction type, you can enable/disable entities that extract data (for example, you might want to disable an entity when that extracted data is not relevant at this moment).  

Work With Entities

  1. On the Home page, select Insights.

  2. Select Entity Management.

  3. Optionally, filter the list of Entity Types (left pane).

  4. Select an entity type.

  5. Optionally, filter the list of Entities (right pane).

  6. Select one or more entities. Or select all entities with the Select All checkbox.

  7. Select Enable or Disable.
    Only enabled entities will be extracted/shown in Document View.
    Single entities can be enabled/disabled with the status button.

  8. View the list of documents that contain the related entity by clicking the blue # of Documents.

  9. Optionally, select the three dots in the top right corner to export the list of enabled/disabled entities to CSV or Excel.

Switch to the Library/Manage Jobs/Consumption Overview (top right corner)





Delete Entities

You can delete an Entity along with the index and facets, by the Entity Type.

  1. On the Home page, select Insights.

  2. Select Entity Management.

  3. Optionally, filter the list of Entity Types (left pane).

  4. Select the entity type you wish to delete.

  5. Click the Delete icon next to the name.

    A confirmation message will appear:

  6. Click OK to continue. A notification message appears.

Entity Type Status

There are two statuses that will be shown.

  • Deleting: Whenever an entity type is in the process of getting deleted.

    Note: Whenever an entity is under deletion with the status beside it showing “Deleting”, the user is not allowed to schedule another job for the same entity until the deletion is completed.

  • Updating: Whenever a new entity type is in the process of getting added.
