Overview:

In search you can create searches on live data and export data directly into ZYLAB ONE review. Prior to using , permissions and custodians must be configured. For more information, see Configure .

Getting Started

  1. Open ZyLab One. Select your Matter.

  2. From the Home page, select the tile.

    The Search window opens.

  3. Click Start New to create a new search.

  4. Enter a Name for the search. This name will be included in the upload name, so being specific can help identification later in your workflow. Entering a description is optional.

  5. Click Continue.

  6. Once the search is created, you will see Volume (GB) and File Count next to the search name. This is your custodian scope volume and file count, which will update as you refine your search.

    Note: Here you see the total amount of data of your selected searches. Currently it will not deduplicate your data (so if documents are present in multiple searches, they will be counted multiple times), but in general they will be deduplicated during processing.

You are now ready to create your search. It will consist of three steps.

  • Discovery Questionnaire: Building your Search

  • Visual Preview: Refining the Search

  • Results Explorer: Viewing the Resulting Files

Step One: Discovery Questionnaire

In this step, you will create the search. Complete each section.

  1. Who is involved: Enter Custodians individually or select from a list. Click Save. This list can be updated later if needed.

  2. Where is the data located: Select from a list of data sources. Select all if unsure.

  3. What is the search about: This is where you can enter your search keywords. Press enter or add a comma between keywords.

    • Click the slider to include files with no text.

    • To add expressions to your search criteria, click .

      Standard query syntax, term searches, proximity searches, and wildcard searches are supported. The operators AND OR NOT, as well as parentheses, to impose an order of operations, are supported.

  4. When were the files last modified, received, or sent: You can select a date range for the search, or check the box to include all dates.

    • Click the slider to include unknown dates.

  5. After all selections, click Save .

  6. To go to Step Two, click Next Step .

Step Two: Visual Preview

In this step, you can review the search results visually. Each section allows you to refine the original search terms.

  1. Insights: This section will identify file exceptions.

  2. People and Sources: Based on the results you can:

    • Click Edit Custodians to delete or add custodians.

    • Click Edit Sources to delete or add data sources.

    • Check Include Participants to display the custodians that were not included in the original search scope, but are added when the box is checked.

      Participant Example: If you only include "Custodian A" in your query and also add "Keyword B", the system will look for any other custodians that used "Keyword B", but are not part of the original search scope. They will be listed as "Participants" when the Include Participants box is checked and will be added to the search scope.

  3. Keywords & Expressions:Click Edit to modify the Keywords or add search expressions to refine the results.

  4. Dates: Click Edit to delete or add date ranges to the search.

    • Click to display different date ranges for review. Click Reset to return to the original view.

  5. After all selections, click Save .

  6. To go to Step Three, click Next Step .

Step Three: Results Explorer

This is the final step. You can preview the resulting files individually.

  1. The documents are presented in a grid view with columns.

    • The column view can be edited by clicking Edit Columns in the far right corner.

  2. Click on the Document ID to open the document preview window. In this view the search keywords will be highlighted.

    Toolbar Controls Include:

    • Back arrow to previous file

    • Zoom view controls

    • Download File

    • Include in Promotion

    • Next File arrow

  3. When the review is completed, click . This will take you back to the search home page.

Promote Data

When the search is finalized, you are ready to promote your data to the review platform.

  1. From the Search Home page, select the searches to send to review. Once selected the Collection button becomes available.

  2. Click Collection. The number indicates how many searches are selected.

  3. The Collection Summary window will open.

    • A search can be removed by clicking the .

    • Click the Back button to return to the Search Home page to modify your collection.

    • The Total to Collect will give you a summary of the size and file count being processed into review.

      Note: Searches can not be edited after promotion.

  4. Click the Send to Review button to promote the data into the review platform. An upload job will be started.

    Note: When the upload job is started, it is connecting to the original source location of the live data to collect. Depending on the size of the data and source, this could take a while to complete and is not instantaneous.

  5. Click Matter>Upload to view the progress of your job. When status is complete, a .CSV download is available.

Review

When the upload is complete you can now review the promoted documents.

  1. From the Home page, select Browse.

  2. Add the Upload Name facet to your view.

  3. Type your search name in the filter to quickly find your upload.

  4. Switch to Document List view to review individual documents.

  5. You must add the column fields to your view.

    • Click Settings>Columns.

    • Check the boxes for the fields.

      Note: The fields must be part of the Matter Template before they will be visible for selection in Columns.
