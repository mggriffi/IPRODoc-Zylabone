Matter Dashboard

After logging into ZyLAB ONE, you will see the Select Matter dashboard. In the Select Matter dashboard all ongoing matters are displayed.

When you receive an access request, the preliminary phase does not change. The controller will first determine whether the access request is specific enough. The controller will then identify the data subject and the grounds for refusal according to the GDPR. If all requirements are met, the access request can be handled. You can create a new matter in ZyLAB ONE.
