Produce Documents

If the user has decided that the documents have been thoroughly filtered, tagged, stored and/or anonymized and/or pseudonymized, the processed documents can be produced. The documents can be produced in either Image file (TIFF or PDF), Native File or Text File.

The production process is started in Document List View. Here, the user must filter the documents to only show the documents that will be processed. This is commonly done by tagging the documents that are ready for production with a Production tag, however other tags can be placed instead of the Production tag as well.


Start a Production

  1. Search for the documents that you want to include in your production.
    You can skip this step, if you want to include all documents from your matter.
  2. In Document List, select the documents that you want to include in your production.
    You can skip this step, if you want to include all documents displayed in Document List.
  3. Keep the default sort order, or customize it.
  4. Now, you are ready to start the Production Wizard. 
    Note: Depending on the permissions you have received, you may not be able to produce.
    • Select the Settings icon  
    • Select Productions.
    • Select Create for Documents or Create for Selection.
      The Production Wizard will appear.

View All Productions

For an overview of all productions:

  • Select the Settings icon 
  • Select Productions.
  • Select Show All Productions.
    The Production Overview will appear.
