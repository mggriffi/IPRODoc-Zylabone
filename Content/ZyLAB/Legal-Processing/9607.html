Edit Global Settings

Conditions

You want to edit the settings that are relevant for all matters.

Instructions

  1. Select the Open Settings icon .
    The Global Settings dialog will appear.

  2. You can define another path for the Shared Storage Location.
  3. You can adjust the Matter Defaults.
    • To configure the default collection settings, click the Collection Settings button. For more information, see Define Custodians, Define Sources and Locations and Define File Groups.
    • To configure the default file groups, click the File Groups button. For more information, see Define File Groups (start with step 4).
    • To configure default sources that can be automatically added when adding a custodian, click the Default Sources button and define one or more default sources. Click Close.
    • To configure default processing rules, click the Processing Rules button. For more information, see Link File and MIME Types to Processors.
    • To configure the default security settings, click the Security button. For more information, see Define Security Settings.
  4. You can define the Index and Web Client Creation Settings and select a Legal Review Template.

    An index will be created based on this index template when a user selects Save & Check-In in a matter.


    Choose from:

    • EDiscovery (index to ZyINDEX/ZyFIND)
      Documents can be viewed and produced in ZyLAB Legal Review, but not reviewed.
    • EDiscoveryLegalReview (index to ZyLAB Legal Review)
      Documents can be reviewed and produced ZyLAB Legal Review.
      Documents are exported to Legal Review. A TBIE index will be created and fields will be stored in the Legal Review database.
  5. Click Save & Close.

Result

You have edited the settings that are relevant for all created matters.
