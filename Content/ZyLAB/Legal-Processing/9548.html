MIME Type

MIME Type matching is an advanced analysis function that classifies a file based on its Multipurpose Internet Mail Extension (MIME) type and its actual content. This is used to determine the type of file irrespective of its file name or extension.

The MIME type database is based on the DROID (Digital Record Object Identification) software and is encoded in the ZyLAB eDiscovery software; updates are periodically available from ZyLAB. Contact ZyLAB or your ZyLAB Reseller for details.

We now offer the possibility to detect different types of PDF (Image, Electronic, Mixed, Portfolio, Embedded Object, Encrypted, etc.).
