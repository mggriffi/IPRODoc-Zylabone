Archive Extractor

The passwords in the Archive Extractors XML configuration file (\\Program Files\ZyLAB\EDiscovery\Processors\ZyLAB.EDiscovery.Processors.ArchiveExtractor.dll.xml) can be configured. A list of passwords can be defined which the Archive extractor will try if it encounters a password protected archive. For example, the configuration could look like this:

<?xml version="1.0" encoding="utf-8" ?>
<zylab>
  <configurations>
    <configuration id="Default">
       <settings>
<ArchivePasswords>
<!--<![CDATA[
        <passwords>
<password value="YourArchivePassword1"/>
<password value="YourArchivePassword2"/>
</passwords>
]]>-->
</ArchivePasswords>
       </settings>
    </configuration>
  </configurations>
</zylab>
