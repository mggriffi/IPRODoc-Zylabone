Exchange Collector

The Exchange Collector collects objects from Exchange Online, Exchange 2010 and Exchange 2013 using Exchange Services. Objects are downloaded as MSG files.

Exchange Online Configuration

Exchange Collector impersonates the accounts listed for collection. Therefore, impersonation must be enabled in Exchange Online.
For Exchange Online, select Manage, go to Roles and Auditing and create a new Administrator Role Group (ImpersonationRole). Add the ApplicationImpersonation and Mail Recipients role. Also, add the ExConAdmin mailbox to the list of members of the ImpersonationRole group.

Exchange 2010 Configuration

For Exchange 2010, execute the following Exchange PowerShell commands to assign the impersonation permissions to Mailbox user ExConAdmin:

New-ManagementRoleAssignment -Name 'ExConAdmin' -Role 'ApplicationImpersonation' -User ExConAdmin@domain.com

Get-MailboxDatabase | Add-ADPermission -User ExConAdmin@domain.com -ExtendedRights Send-As,Receive-As

Exchange 2013 Configuration

Impersonation must be enabled in Exchange 2013.
For Exchange 2013, select Manage, go to Roles and Auditing and create a new Administrator Role Group (ImpersonationRole). Add the ApplicationImpersonation and Mail Recipients role. Also, add the ExConAdmin mailbox to the list of members of the ImpersonationRole group.
Execute the following Exchange PowerShell commands to assign the impersonation permissions to Mailbox user ExConAdmin:

New-ManagementRoleAssignment -Name 'ExConAdmin' -Role 'ApplicationImpersonation' -User ExConAdmin@domain.com

Get-MailboxDatabase | Add-ADPermission -User ExConAdmin@domain.com -ExtendedRights Send-As,Receive-As

  • Settings

    Exchange Version
    Select the correct Exchange Version.

    Exchange Web Service URL
    Exchange Web Services URL is optional if the Autodiscover service has been configured correctly. Using Autodiscover service is necessary in environments with multiple Client Access Servers.

    Login Account
    Define the (ExCon Admin) account.

    Login Password
    Enter the correct password.

    Login Domain
    You have the option to define the domain for authentication.

    Mailbox accounts
    Click Add to select Mailbox accounts that must be archived.

    Folders
    Select folders from which to download messages.

    Skip Journal Body
    Choose Skip Journal Body to skip journal items and only archive the attachments from journal items (as in Archiving Services, journal body text will be stored in emxjrnl field).
    Note: Microsoft does not support journaling to Exchange Online accounts.

    Search Filter (Optional)
    Define Subject, Body and/or Date Range to filter the messages you collect.

    Note: the date range is not dynamically adjusted. So, when you choose to collect everything until today (inclusive) and want to rerun the same collection action the next day, it will collect everything until yesterday (inclusive).

  • File Types to Collect
    Select the appropriate File Groups. To create new File Groups, click Add New.
    For more information on how to define File Groups, see Define File Groups.
