ZyLAB Index Collector

The ZyLAB Index Collector collects files over http(s) using the ZyWebserver WebClient (ZyNET.exe).


  • Settings

    Server
    The computername of the ZyWebServer Client. For example, digital2.

    Client
    Client name. For example, Email Client.

    Indexes
    Long name(s) of Zy-index(es) (pipe separated).

    User
    The Zy-user username (for example, anonymous)

    Password
    The Zy-user password (for example, anonymous)

    NT User
    The NT username.

    NT Password
    The NT user password.

    Use SSL
    Use the Secure Socket Layer of HTTP.

    Full Text
    The key word(s) in document to search.

    Date Range
    The between date range of created/uploaded documents in ZyINDEX.

    The date range is not dynamically adjusted. For example, when you choose to collect everything until today (inclusive) and want to rerun the same collection action the next day, it will collect everything until yesterday (inclusive).

  • File Types to Collect
    Select the appropriate File Groups. To create new File Groups, click Add New.
    For more information on how to define File Groups, see Define File Groups.
