Define Security Settings

Conditions

You want to define the security settings for each user.

Instructions

  1. ZyLAB Legal Processing is open.
  2. Select the Open Settings icon .
    The Global Settings dialog will appear.
  3. Select the Security button.
    The Edit Default Security Settings dialog will appear.

  4. To add a user, click - in the User View tab - the Add Users button.
    • To delete a user, select that user and click the Delete button.
  5. In the Add User(s) dialog, define the name of a user and click the Resolve button.

  6. When the user is found, click Add.
  7. Repeat step 5 and 6 to add more users.
  8. Click Next.
  9. Define the roles (levels of access) for the user(s) that you selected.

  10. Click Finish.
    • Alternatively, you can select a user in the Edit Default Security Settings dialog and add assign roles or remove role assignments.
  11. Click Close.

Result

You have defined security settings for each user.

Note

To define the roles (levels of access), click the Open role manager button. In the Global Roles dialog you can (de)select the permissions of a role.
Select the Role View tab to edit roles, add users to roles or remove users from roles.


Also, you can create a new role (select the Create new role button, define the New Role name, select the permissions and click Done).
