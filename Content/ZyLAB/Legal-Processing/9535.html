View or Filter Actions

The actions for all the matters are visible in the Actions Tab. You can view and filter actions.

View Actions

  • Sort actions by selecting a Column Header.
  • Drag and drop a Column's header to change the order of the grid.
  • To view the Chain of Custody double click on an action.

  • Move to the first/last or next/previous page with the arrow buttons at the bottom left corner of the screen. Alternatively, directly enter a page number.

  • Define the Page Size (number of actions displayed on one page) at the bottom left corner of the screen.
  • Total number of records/actions is displayed at the bottom left corner of the screen.

Filter Actions

  • Select the Pencil icon to open the Filter Editor.

  • To define the operators, select the red items. Choose from And, Or, NotAnd, NotOr, Add Condition, Add Group and Clear All.
  • To define the column fields, select the blue items. Choose from Id, Status, Matter ID, Matter Name, Job Id, Priority, Custodian, Source, Location, Performer, Performer Configuration, Scheduled Time, Start Time, Finished Time.
  • To define the conditions, select the green items. Choose from Equals, Does not equal, Is greater than, Is greater than or equal to, Is less than, Is less than or equal to, Is between, Is not between, Is blank, Is not blank, Is any of, Is none of.
  • Double click on <enter a value> to define a value.
  • To add a new condition to a group, select the green plus sign.
  • To remove a condition, select the red cross sign.


  • You can deselect (use the checkbox) or remove (use Clear Filter icon) a filter and see the actions for all the matters.
