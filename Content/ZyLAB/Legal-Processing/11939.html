Filtering Report

Categories of filtered documents are: Documents Excluded due to No Processing Rule Assigned, NSRL Exclusions. This reporting works for all Legal Processing collectors.

No Processing Rule Assigned

If no processing rule is associated with a document, then that document is skipped during processing. In the Filtering Report you can view which files existed in the original collection, but were excluded.

NSRL Exclusions

In the Filtering Report you can view which files existed in the original collection, but were excluded due to NSRL (or NIST) exclusions.
