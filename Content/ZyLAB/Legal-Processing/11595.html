Configure Text Extraction

Configure Text Extraction to determine what information should be extracted and made available to users. What users will find, depends on the settings in the configuration file. The configuration should be done before matter creation in Legal Processing.

Text extraction is configured per index. Each TBIE index in ZyLAB Information Management Platform 6.5 and higher contains an index-text.cfg file. This file can be changed just after index creation (before any documents are indexed) or in the index template.

Note: Do not change this file (index-text.cfg) after documents were added to the index.

Instructions

  1. Open the configuration file of Text Extraction.
    The default location of the configuration file is ZyLAB Data\TBIE Indexes\INDEX_NAME\index-text.cfg
  2. For the reader(s) being used, for example, the OITContentStream (text reader to extract electronic files), define the file types (extensions) that need to be extracted and the email headers users will be able to see.

    It is recommended to keep the default encoding, textlayout, textseparator and normalization settings to prevent unexpected behavior.
