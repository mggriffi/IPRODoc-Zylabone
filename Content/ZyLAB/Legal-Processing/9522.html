Define File Groups

Conditions

Define which file types of each location should be collected during processing. You can create/select different custom made file groups for each location.

Instructions

  1. ZyLAB Legal Processing is open.
  2. If the relevant matter is not yet opened, search and open it. The last created matter is shown first.
    You can search for a matter via the tabs on the left side of the screen:

    • All
      All created matters will be shown.
    • Active
      Active matters are all opened and being worked on.
    • Inactive
      Inactive matters are all closed. Nobody is working on them.
    • Checked-out
      Checked-out matters can be open (active) or closed (inactive), but only the person who has checked a matter out can open it.
    • Checked-out by me
      Checked-out by my matters are matters from the logged-in user. They can be open (active) or closed (inactive). Only the logged-in user can open and work on them.
  3. Select the File Groups tab



    or

    In the Collection tab, select the checkbox Show File Groups Editor. If you define File Groups via this option, you can link them directly to locations: Select a Location and click Add Group (continue with step 5 below).

  4. In the File Groups tab, select Add.



    The Add File Criteria dialog appears.

  5. Define a New Group name.
  6. Click Add.
  7. Select Criteria to Add. Choose from
    • Wildcard
      Define a file type (for example, *.xxx) and click Add. It will be added to the Group. Repeat to add more file types.
    • MIME Type
      Search and select a MIME Type from the list. Click Add. It will be added to the Group. Repeat to add more MIME Types.
  8. Click Close.
  9. To edit or remove a Group, select it and click the Edit or Remove button.
  10. To remove a type from the group, select the group and the type and click the Remove button.
  11. To add a Wildcard or MIME Type to the Group, select the group and click Add Wildcard or Add MIME Type. The Add File Criteria dialog will appear.
  12. Click Save (continue editing or click Cancel to exit) or Save & Check-In (the Matter will become Read-only, click Close to exit or Check-out & Edit to continue editing).

Result

You have defined File Groups.
