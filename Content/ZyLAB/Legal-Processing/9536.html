Create Reports



Instructions

  1. To create a report, select the Reports tab.
  2. Via the tab on the left side of the screen, select a report.
    You can select from five predefined reports:
    • Matter Actions Report
    • Data Per Matter
    • Matter Job Time Report
    • Collection Errors
    • Failed Actions Report
    • Filtering Report

      Or you can create a custom report, based on the:

    • Matter Pivot Report
    • Matter Actions Pivot Report
  3. Select a matter.
  4. Click Generate Report.
  5. You can adjust the report by moving column fields around. The column fields located at the top of the screen are not added to the final report. You have to drag and drop them to add them. The order of the column fields can be changed too.

  6. Click Refresh Data to update the information.

    When data is saved or exported during some matter job running, only the current view data (not refreshed) is exported or saved.
