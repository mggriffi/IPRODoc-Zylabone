Sort

  • Sort on one column: Click on a column header. Click again to switch between ascending/descending.

  • Sort on multiple columns: Ctrl+Click successive column headers. Ctrl+Click again to switch between ascending/descending.

    Tip: The sorting order is displayed above the document list. For more information—see Sort.

  • Change column location: Drag&drop column headers in the prefered order.

  • Change column width: Select vertical border and drag&drop. 

    Return to evenly distributed column widths: Click 'Reset column sizes' icon.
