Auto Redactions

Automatically find and redact keywords, phrases, and personal information (names, social security numbers, credit card numbers, email addresses, and more).

Add New Auto Redaction Rule

  1. Select   
  2. Define a Name.
  3. Specify the Redaction Text and/or Code Set/Code Entry.
    (Optional) Add a motivation.
  4. Select a Fill Color and a Font Color.
    These can be solid or transparant.
  5. Select a Font Family and a Font size.

  6. Specify one or more queries.
    In the top right corner, you can choose from Full-Text Search, Insights Entity Types and PTM Entity Types (deprecated).

     

    Note I: Insights Entity Types are only shown when Insights is enabled.

    Note II: PTM Entity Types (deprecated) will only be shown/used when old PTM tables are present. It is no longer necessary to use prefixes (like PTM_AUTOREDACTION or ENTITY_AUTOREDACTION).

    For samples—see  Example Queries. For more information—see ZyLAB One Search Language Guide.

  7. Define the Redaction Filter. For more information—see Auto Redactions.
  8. Click Apply.
    The new rule is added to the left-hand pane.
    The redactions will be applied immediately on all affected documents. Adjacent hits that belong together will be merged.
