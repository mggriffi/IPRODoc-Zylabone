Validation Set

Create a Validation Set (Test Set) to determine the performance of the Classifier. 

  • To include a Validation Set, select the Use Validation Set option when adding a new project to Assisted Review.

    Note: You cannot add a Validation Set to an existing project.

  • If the size of the Validation Set is insufficient (you do not have enough random documents to estimate the number of responsive documents in the project), then you can extend the Validation Set. Select the Extend Set button.
