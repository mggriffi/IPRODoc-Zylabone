Initial Training Sets

Determine the size of the Initial Training Set, used to start training the Classifier), as based on:

  • Random Set

    Select a batch size from between 2 and 1000 documents.

  • Query-Based Set

    Select a search query that was saved in ZyLAB One.

  • Based on Existing Tags

    Select saved search queries for Responsive and Not Responsive documents.

    System will take the batch size of random documents of the results of this query.
