Search Hits Preview

Use Search Hits Preview to view the context of a hit, without opening the file. Adjust the Search Hits Preview settings:

  1. Select the Search Hits Preview Settings icon:

    • Enter the Maximum Number of Hits shown for one file.

      (Example) If the Maximum Number of Hits is set to 3, then the file might contain more than 3 hits, but no more than 3 hits are shown in preview.

    • Enter the Maximum Number of Words Before and After a hit.

      (Example) If the Maximum Number of Words is set to 5, then no more than 5 words preceding and 5 words following the hit are shown in preview.
