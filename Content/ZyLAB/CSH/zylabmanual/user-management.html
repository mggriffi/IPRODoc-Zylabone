User Management

Add Users/Groups        

Select  to (re-)add users/groups from the Active Directory.

  1. Select User(s)/Group(s) from the relevant tabs.
    Filter if needed.

     

    • For a selected group, select the Related Users tab for an overview of all users in this group. 

  2. Assign/remove a Global role via the Global Roles tab. 

    Assign/remove a Matter role via the Matter Roles tab.

  3. For an overview of all assigned roles to a user or group, select the Overview tab.

Delete User/Group  

Delete a user/group from User Management.

 

Assign Role to User/Group  

  • Global roles assigned to users/groups apply to all matters. 

  • Matter roles assigned to users/groups apply to one specific matter.

    Important: Global roles override Matter roles.
