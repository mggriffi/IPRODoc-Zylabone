Delimited File Settings

Select the Concordance Format.

There are three default export methods in Concordance (see list below). Select one of them and the Comma, Quote and New Line fields will be automatically set to the right value. 

If you have set a Custom Format while exporting, set the Format option to custom and define the Comma, Quote and New Line yourself. Please make sure to provide the same settings as it was exported with.

Default export methods in Concordance:

  • Concordance default
    Using the .DAT format (with as Comma (020), as Quote Þ (254), and as New Line ® (174))
  • Comma delimited (CSV)
  • Tab delimited 

Select the Date Format.

Date format should be set to the same format as the file is exported in from concordance.
