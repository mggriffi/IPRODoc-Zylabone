Settings

Though the Settings of a load file can be adjusted, it is recommended to keep the default settings, because these settings are used by the other application to import the data. They should only be changed when the data actually contains these characters in the metadata (very uncommon).

  • CSV
    Column, New Line, Quotation Mark, Multi-Value, Encoding, Date Format
  • XLSX
    Column, New Line, Quotation Mark, Multi-Value, Encoding, Date Format
  • Summation
    Encoding, Date Format
  • Opticon/Concordance
    • .DAT File Settings
      Column, New Line, Quotation Mark, Multi-Value, Encoding, Date Format
    • .OPT File Settings
      Opt Separator, Opt Volume Name, Opt Encoding
  • EDRM XML 1.2
    Locale, Description, Case ID, Date Format
  • IPRO
    Volume Name

Field Mapping

For IPRO only images will be produced; no fields have to be mapped.

For all other load format:

  • Select a source field you want map in the panel on the left.
  • Select the Add Field arrow to add it.

  • Select a field type from the dropdown list box.
  • Define a Map To Field.
  • Mapped fields can be removed by clicking the trashcan icon.
  • Mapped fields can be moved up and down with the Move Up and Move Down buttons on the right.
