Configure LIVE EDA Settings

Select the custodians to be included in the scope of your LIVE EDA matter and case.

Select Custodian(s)

  1. From the left-hand pane, select custodian(s) by clicking on the corresponding checkbox(es).

    • Click Select All to select all custodians.

    • Use Search to find and select (a) specific custodian(s).

  2. Click on the arrows > < to add or remove custodian(s) to or from the Selected Custodian list.
    Custodian(s) added to the Selected Custodians list, are the custodian(s) available for selection in the LIVE EDA investigation(s) created.

  3. When finished, click Confirm (located bottom-right).

Enable/Disable Selected Custodians

Enable or disable the custodian selection previously configured and confirmed.

  • (Default) When enabled, the custodian(s) added to the Selected Custodian list become available in Live Sources, where you create the LIVE EDA investigation(s). You will be able to select any of the available custodians in the LIVE EDA investigation(s) created.

  • (Optional) When disabled, the custodian(s) added to the Selected Custodian list become unavailable for use. Live Sources, where you create the LIVE EDA investigation(s), becomes inaccessible.

    Note: You cannot disable Custodian Configuration if there are any active promotions. Promotions that are 'in-progress' or 'partially successful' need to first be canceled from within the corresponding LIVE EDA investigation.
