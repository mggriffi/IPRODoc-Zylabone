Entity Management

View a list of all entity types. Per entity type, you can enable/disable entities that extract data (for example, you might want to disable an entity when that extracted data is not relevant at this moment). For more information—see Entity Management.

Work With Entities

  1. Optionally, filter the list of Entity Types (left pane).

  2. Select an entity type.

  3. Optionally, filter the list of Entities (right pane).

  4. Select one or more entities. Or select all entities with the Select All checkbox.

  5. Select Enable or Disable.

    Only enabled entities will be extracted/shown in Document View.

    Single entities can be enabled/disabled with the status button.

  6. View the list of documents that contain the related entity by clicking the blue # of Documents.

  7. Optionally, select the three dots in the top right corner to export the list of enabled/disabled entities to CSV or Excel.

Delete Entities

You can delete an Entity along with the index and facets, by the Entity Type.

  1. Optionally, filter the list of Entity Types (left pane).

  2. Select the entity type you wish to delete.

  3. Click the Delete icon next to the name.

    A confirmation message will appear:

  4. Click OK to continue. A notification message appears.

Entity Type Status

There are two statuses that will be shown.

  • Deleting: Whenever an entity type is in the process of getting deleted.

  • Updating: Whenever a new entity type is in the process of getting added.
