Create Production

Follow all steps of the Production Wizard to start a production.

= Active step that is being edited.   

 = Inactive step that is not yet edited.

 = Completed step that can be edited again.

  = Error. Resolve before continuing with next step.

1. Define Production

In step one, select a Production Template, define the Name and define the Bates Label Settings.

View all production template settings in the right pane.

For more information, click the help icon next to Definition and/or refer to Define Production.

2. Image Settings

In step two, define the Produce subtab. When all documents should be produced as images, also define the Burn-in Fields, Redactions and Placeholders subtab. When no documents should be produced as images, continue with Native Settings.         

View the production preview in the right pane.

For more information, click the help icons in the subtabs and/or refer to Image Settings.

3. Native Settings

In step three, define if all, some or no documents should be produced as natives. Also define the Naming and the Folder Properties.

View the production preview in the right pane.

For more information—refer to Native Settings.

4. Text Settings

In step four, define if all or no documents should be produced as text. Also define the Naming and the Folder Properties.

View the production preview in the right pane.

For more information—refer to Text Settings.

5. Load File

In step five, select one or more load file formats.

For more information, click the help icons and/or refer to Load File.

6. Production Summary

In step six, view a summary of all your (modified) production template settings.

Click the Print button to print the summary (to a printer or to PDF).

For more information—refer to Production Summary.
