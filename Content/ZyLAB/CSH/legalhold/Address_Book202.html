Address Book

The Address Book represents all potential contacts of any hold. It also is a combination of the company’s employee directory (usually provided by Microsoft Entra ID) as well as any additional contacts that are manually entered (or imported) for people outside of the company. 

Multiple contacts can be imported all at once via the settings  icon. 

View, filter, sort, create, edit, delete, import or export contacts.

  • View created contacts and filter/sort them via the columns.

    Note: Filtering/sorting works the same on all pages (example below is of the Hold Management page).

  • Via the Settings icon  synchronize contacts, import contacts, add/remove columns or export the Address Book View as CSV, Excel or PDF.

  • Edit a contact via the Edit Contact icon .

  • Delete a single contact via the Delete Contact icon . 

  • Delete multiple contacts by selecting them and clicking the Delete selected Contacts icon . 

  • Update the Address Book View with the Refresh icon .

Create Contact

  1. Select Create Contact.

  2. Define the Contact Details.

  3. Click Save.
