Create Questionnaire Template

  1. Define the Name (mandatory).

  2. In the Survey Designer tab, select questions from the Toolbox to add them to the (first) page of the questionnaire.

  3. The properties of each question can be edited.

  4. In the Test Survey tab, check the end result.

  5. In the Survey Logic tab, add rules to customize the flow of the survey.

  6. Click Save.
