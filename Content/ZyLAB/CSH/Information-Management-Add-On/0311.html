Archives
Add, edit and/or close an archive.
Go, via the menu, to Settings > Archives.

Group by dragging one or more column headers above the grid. You can also move them back to the grid.
Sort (ascending/descending) by clicking in the column header once.

Select  for more information per shown result.
Select  to hide this information again.

New Archive
Select  (New) to add a new archive.
Fill out all fields and select Save.

Edit Archive
Select  (Edit) next to an archive to edit it.
Adjust the fields and select Save.

Close Archive
Select  (Close) next to an archive to close it.
Confirm with OK.
