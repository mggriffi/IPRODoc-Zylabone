List Fields

List Fields
Add a list field to a category, edit and/or delete a list field.
List fields and their values are used when creating for example a record.
Users will be able to select one or more values from a list (mulit or single select).
The default categories (to which list fields can be added) can be extended via an xml file. For more information, please contact your Administrator or ZyLAB.

Group by dragging one or more column headers above the grid. You can also move them back to the grid.
Sort (ascending/descending) by clicking in the column header once.

Select  for more information per shown result.
Select  to hide this information again.

New List Fields
Select  (New) to add a new list field.
Enter a value and select Update.
A default value will be automatically filled out when creating for example a record.

Edit List Field
Select  (Edit) next to a list field to edit it.
Adjust the value and select Update.

Delete List Field
Select  (Delete) next to a list field to delete it.
Confirm with OK.
