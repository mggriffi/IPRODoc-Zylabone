Entity Management
View a list of all entity types. Per entity type, you can enable/disable entities that extract data (for example, you might want to disable an entity when that extracted data is not relevant at this moment).  

Instructions
  1. Optionally, filter the list of entity types (left pane).
  2. Select an entity type.
  3. A list of entities appears (right pane). This list can be filtered.
  4. Select one or more entities. Or select all entities with the Select All checkbox.


  5. Select Enable or Disable.
    Only enabled entities will be extracted.
    Single entities can be enabled/disabled with the status button.


  6. View the list of documents that contain the related entity by clicking the blue # of Documents.
  7. Optionally, select the three dots in the top right corner to export the list of enabled/disabled entities to CSV or Excel.
