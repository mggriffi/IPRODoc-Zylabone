Create Batch Download
Batch Downloads offers the same functionality as Mini-Productions in ZyLAB Legal Review (version 3.11/3.12).

A batch download will be created for all documents or a selection from the Document List.

  • Adjust the Name if needed.
  • To produce numbered PDF files, make sure the checkbox Produce Document Numbers is selected.
    If this checkbox is not selected, the original name of the document is used.
    • Define the Document Numbers Settings. This is how the PDF files will be numbered.
  • Click Create.
    A summary is shown.
  • Click Done.

For more information, please refer to Batch Downloads.
