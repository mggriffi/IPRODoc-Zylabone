Main Fields
Define how fields will be linked. Select a field name, as used in the delimited file and the image load file, for each field type (on the left). 
The (content of these) fields will be added as meta data.
  • Document ID *
    Used to link a field from the dat file with the opt file.
    * Mandatory field type to define
  • Link to Parent
    Images (ZyXMLTiff files) are linked to images.
  • Link to Native
    Natives are linked to natives.
  • Link to Text
    Text files are linked to text files.
  • Attachment List
    If parent documents and attachments have been broken out into separate records, there may be a Begin and End Attachment field or Attachment List field in the database for each record.

  • Attachment End
