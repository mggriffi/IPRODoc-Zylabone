ZyLAB Collector
The ZyLAB Collector enables you to copy (export) data from one matter to another. Documents and metadata (fields) will be collected and exported.

Instructions
  1. Define 
    • Sources
      Select the source matter(s) you want to collect data from.
    • Family Expansion
      Select the mode in which documents must be collected.
      For now, only 'None (only matching documents)' is available.
    • Impersonate Service User
      It is advised to keep this setting enabled.
    • Set the Custodian and Source for all locations
      Define (select or add) the custodian and source.
  2. Click Next.
  3. Define the Mapped and/or Not Mapped fields.
  4. Select Start Data Collection.
