Reports
Reports keep the process (of uploading, processing, reviewing, etc) transparent.  
There are multiple options to create/view reports:
  • Export to CSV
    To create a report on your Uploads or the documents in the Document List.
    The .csv (comma separated values) file type can be opened in a spreadsheet program like Microsoft Excel.
  • Matter Overview
    To view all available information on the documents in your matter. 
  • Reports

    Via the menu option Reports, create one or more of the available reports.

    For each report the following options are available:


    • Page navigation options
      First page, previous page, specific page, next page, last page.
    • Refresh the contents of the report
    • Go back
    • Enlarge or reduce the size of the report page
    • Save to Word, Excel, Powerpoint, PDF, TIFF, MHTML, CSV, XML or data feed
    • Print the report
    • Search for content in the report
