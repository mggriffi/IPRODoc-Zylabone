Create Issue from Topic
Define a new issue (defines the information need) based on a topic (cluster of words) within the project. New issues will appear in the Assisted Review Progress tab.
  • Topic Label
    Topic on which you want to base your new issue.
  • Topic Id
    Identification number of the topic.
  • Issue Name
    The new issue name.
  • Issue Description
    A description of the new issue.
  • Batch Size
    Size of the Initial Training Set. Select a value between 2 and 1000.
