Roles

A role describes the rights a user has.
There are 4 roles: System Administrator, Functional Administrator, Writer and Reader.
Go, via the menu, to Settings > Roles. 

Edit Role    
Select  (Edit) next to a role to adjust the users connected to it. 
Edit the Active Directory-group(s) connected with this role. Select Save.
