Archiving method

There are two archiving methods:

  • Scan
    Register paper documents with this archiving method. 
    Define metadata during registration. The metadata depends on the selected record type.
    Print the defined metadata and scan it together with (on top of) the paper document you want to archive. After scanning, the metadata is added to the paper document (which is now electronic and searchable).
  • Web Upload
    Register electronic documents with this archiving method.
    Define metadata during registration. The metadata depends on the selected record type. The metadata is added to the electronic document.

    This archiving method is also used when archiving an email from Microsoft Outlook:
    • After you selected 'Archive emails to ZyLAB', select 'Emails and attachment(s)', 'Email only', or 'Attachment(s) only'.
    • Select an archive. 
    • Select a record type.
    • Register.
      Depending on the selected record type define a number of fields.
      For more information, see Records Overview.
