Home

Select Home in the menu to go to your start page. Your start page depends on the number of archives you have.


Records Overview page with one archive

With 1 archive the Records Overview page is your start page.
On the Records Overview page you register new records (documents). Make sure each new record is stored in the correct file and is archived properly.
Existing records can be organized (grouped, sorted and/or filtered) or actions can be performed (edit, delete, view, copy). You can also export records.


Overview page with two or more archives

With 2 or more archives the Overview page with all archives is your start page.
First select an archive before you start registering documents on the Records Overview page (see above).
