File

When creating a new record, you can also select the file. In the selected file, binaries are stored.


A file can, for example, contain all information of an employee for human resources, or it can contain all information of a client or account for a sales department.

Every file has its own metadata. For an employee that can be his name and employee number. 

This metadata will be linked to the record.

 

Files are created, edited and deleted via Settings > Files.

First select an archive if you do not see this Settings option.
