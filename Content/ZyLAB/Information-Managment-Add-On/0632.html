Users

Add, edit and/or delete users.
Go, via the menu, to Settings > Users.

Group by dragging one or more column headers above the grid. You can also move them back to the grid.
Sort (ascending/descending) by clicking on a column header once.

Select  for more information per shown result.
Select  to hide this information again.

New user
Select  (New) to add a new user.
Fill out all fields and select Save.

Edit user
Select  (Edit) next to a user to edit it.
Adjust the fields and select Save.

Delete user
Select  (Delete) next to a user to delete it.
Confirm with OK.
