Installation ZyLAB Workflow

Conditions

First, you have installed ZyLAB Information Management Platform. Continue with installing ZyLAB Workflow.

Instructions

  1. Start the ZyLAB Workflow Installer file. The Preparing to Install screen appears.
  1. The Welcome screen appears. Click Next.

  2. Accept the terms in the license agreement. Click Next.

  3. If necessary change the destination folder. Click Next.
  4. Click Install to start installation.

  5. Monitor the installation process.

  6. Click Finish when the installation process is complete.

Result

You have installed ZyLAB Workflow. Continue with the configuration of ZyLAB Workflow.
