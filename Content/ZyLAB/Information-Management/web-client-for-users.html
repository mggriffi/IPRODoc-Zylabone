Web Client for Users

View, Download and/or Print the ZyLAB Web Client for Users Manual

The ZyLAB Web Client uses ZyLAB scanning and full-text retrieval technology to provide transparent Internet or intranet access to scanned and indexed information. You can access the ZyLAB Web Client with any Internet browser and search for documents that contain the information you are looking for.

In ZyLAB Web Client, you can organize, group, print, upload and download documents from multiple indexes using all the features that make ZyFIND so powerful.

Using the ZyLAB Web Client, you can easily share information throughout your company even if you have offices in different locations. All you have to do is start your Internet browser and access the URL of your archive. The intuitive and fully customizable interface allows you and your colleagues to find what you are looking for.
