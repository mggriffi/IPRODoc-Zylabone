Service Utility Tool

View, Download and/or Print the ZyLAB Service Utility Tool Manual

The ZyLAB Service Utility command line tool (ZyLAB.Platform.ServiceUtil.exe) is a part of the TBIEService solution and enables you to communicate with the following services (with default end point addresses):

  • Indexing (TBIE service)
    net.tcp://localhost:4112/IndexingService
  • PlatformSDK
    net.tcp://localhost:4115/ZyIMAGE
  • Content Access (CAS)
    net.tcp://localhost:4112/ContentAccess
  • Cluster
    net.tcp://localhost:4115/Cluster
  • Cluster Management
    net.tcp://localhost:4115/ClusterManagement

Use the tool to create/delete/rebuild/optimize indexes, search, retrieve text of the document from the index, and diagnose problems with platform services.

It can be found at \\Program Files\ZyLAB\Information Management Platform\Services\Tools
