General Search Language Guide

View, Download and/or Print the General Search Language Guide

In the ZyLAB General Search Language Guide, we explain how to use the ZyLAB search language to search for one or more keywords within a data set.

The data set usually contains files in many forms. Not only the keywords used in these text, image or audio files, but also the information about these files (the metadata), can and should be searched.

This information is also available in the online ZyLAB ONE Search Language Guide.
