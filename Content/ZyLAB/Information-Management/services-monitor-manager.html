Services Monitor Manager

View, Download and/or Print the Service Utility Tool Manual

The ZyLAB Services Monitor Manager monitors all current ZyLAB services and restarts them if the services are stopped unintended. Services can be started on different machines.

Installation and Configuration

The ZyLAB Services Monitor Manager consists of three layers:

  1. Services Monitor Manager
    Layer that controls from one place on the network the performance on several computers.
  2. Services Monitor
    Layer on which the services run.
  3. Services
    • ZySCANService
    • ZyTIMERService
