Reviewing Documents

Take a Data Driven Approach

Traditionally within competition investigations, external third-party data scientists and other technical professional would be brought in to assist with the competition investigations and gathering of information. ZyLAB has circumnavigated that issue by fully automating this process.

As we can no longer read through all documents from start to end, we take a more data focused approach:

  1. We make all data fully searchable (this is what we call Deep Processing).
  2. We identify and remove Legally Privilege Protected data (LPP).
  3. We identify and remove all data of which we are very certain it is not relevant (NR) such as newsletters, spam, personal email, etc.
  4. We identify, tag and order all data of which we are certain that it is relevant and related to the case using a variety of smart search tooling.
  5. Use machine learning to analyze the rest of the data: We now have a certain amount of data left that does not belong in any of the above categories (step 2 to 4). What we can now do, is teach the computer (using machine learning) what is not relevant (#3) and what is relevant (#4) and let the computer automatically classify the remaining data using these classifiers.

Using this approach, we can in-fact analyze all data, without the need to read everything. This is a much more effective approach, which is not only allowed by courts around the world, but there are more and more courts and regulatory agencies who recommend or even mandate the above approach.
