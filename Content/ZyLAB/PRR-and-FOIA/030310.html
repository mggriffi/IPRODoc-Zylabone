Report

Whether you need to provide the requestor with a report of all withheld documents, all redacted documents, all disclosed documents, or you need a Vaughn Index, ZyLAB ONE makes it easy.

Creating a Custom Report

Custom reports are created from the Document List View. The first step is to filter for the data to be reported on, i.e. all exempt or privileged documents. Reports are configured by adding the fields (columns) containing the information required in your report to the document grid, sort the data as applicable, and select Reports from the menu options.


Tip: Select the check box 'Include Redaction Information' if you want to include redaction information in your report (on which page, who redacted it and when it was redacted, redaction code and justification). Once you have created a custom report, make sure you save it as a View so that you can reuse it at any time without having to re-configure it.
